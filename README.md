## ðŸ—‚ï¸ Problem-Solving Framework

Use this for EVERY problem:

### 1. UNDERSTAND (5 mins)
- Restate problem in own words
- Clarify inputs/outputs
- Ask edge cases

### 2. PLAN (10 mins)
- Identify pattern
- Sketch approach
- Analyze time/space complexity

### 3. IMPLEMENT (20 mins)
- Write clean code
- Test with example

### 4. OPTIMIZE (10 mins)
- Can you do better?
- Review solution approaches

### 5. REFLECT (5 mins)
- What did you learn?
- Similar problems?
